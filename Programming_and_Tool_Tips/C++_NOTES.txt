====== C++ NOTES ======
Created Wednesday 24 February 2016

Course Summary:
* Avoid manual mangement
	* solid objets, RAII
	* smart pointers
* use lambdas
* use standard containers and algorithms
* embrace move semantics
	* especially, think about it when creating your own classes (do I want it to be moveable?)
* consider the PIMPL idiom
* stop writing C with classes
