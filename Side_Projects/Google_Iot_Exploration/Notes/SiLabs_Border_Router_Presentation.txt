====== SiLabs Border Router Presentation ======
Created Wednesday 02 March 2016

* Thread cannot work with older Ember Znet Desktop: needs [[http://www.silabs.com/products/mcu/Pages/simplicity-studio.aspx|Simplicity Studio]]
* Demo setup:
	* Border router: a Rasperry Pi board installed with Thread IP Driver and Border Router Application
	* EM35x Ember development kit (needs EM358 chip for Thread)
	* Zigbee USB stick (programmable with ICE)
	* Software?
* Border router web server is accessed via its IP (for demo: 192.168.42.1)
	* Can discover and clear devices (in theory)
